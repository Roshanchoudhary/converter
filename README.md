# 🔄 Devanagari ⇄ Tirhuta Script Converter

Convert between Devanagari and Tirhuta (Maithili) scripts seamlessly.

## ✨ Features
- Bidirectional conversion (Devanagari ↔ Tirhuta)
- Auto-detection of script
- Copy to clipboard
- Swap input/output
- Character and word counting
- Example sentences for quick testing
- Responsive design for all devices
