# 🔄 Devanagari ⇄ Tirhuta Script Converter

[![Deployed on Cloudflare](https://img.shields.io/badge/Deployed%20on-Cloudflare%20Pages-F38020?logo=cloudflare)](https://pages.cloudflare.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Convert between Devanagari and Tirhuta (Maithili) scripts seamlessly with this powerful web-based converter.

## ✨ Features

- 🔄 **Bidirectional Conversion**: Convert Devanagari ↔ Tirhuta instantly
- 🤖 **Auto Detection**: Automatically detects the script
- 📋 **Copy to Clipboard**: One-click copy functionality
- 🔀 **Swap Input/Output**: Easily switch between scripts
- 📊 **Real-time Stats**: Character, word, and line counting
- 📚 **Example Sentences**: Quick access to common phrases
- 💾 **Download Output**: Save converted text as .txt file
- ⌨️ **Keyboard Shortcuts**: Ctrl+Enter to convert, Ctrl+Shift+S to swap
- 📱 **Responsive Design**: Works on all devices
- 🌐 **Cloudflare Powered**: Fast, global deployment
